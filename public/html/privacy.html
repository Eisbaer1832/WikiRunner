Privacy Policy

Which data is stored locally?

  • Username
  • Game History (if enabled)
  • Preferences like the selected Wikirunner Server instance

Why is this data stored?

Storing your preferences enables you to have a persistent app experience, even across device reboots. Nobody likes having to type in their password every time they open an app.

Which data is collected externally?

Our servers don't store any data permanently. Usernames are only stored for the runtime of the game.

Additional data collection may occur through the ingame Wikipedia webview. We are not associated with the Wikimedia Foundation and therefore can not disclose which data is collected by them.

Which permissions are used?

Wikirunner uses the network permission in order to establish a connection with the game server, as well as Wikipedia
